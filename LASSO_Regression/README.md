# Housing Price Prediction with LASSO Regression

## Overview
This project explores how housing characteristics influence property prices
using the Boston Housing dataset. The analysis focuses on feature engineering,
regularized regression, and model evaluation to identify the most relevant
variables affecting housing value.

## Objectives

• Identify the most important features affecting housing prices  
• Apply LASSO regression for feature selection  
• Evaluate model performance using regression metrics  
• Perform residual diagnostics to validate model assumptions  

## Dataset

Boston Housing Dataset containing 506 observations and 13 original features
related to crime rate, pollution levels, accessibility, and socioeconomic factors.

Additional engineered features were created to enrich the analysis.

## Methods Used

- Exploratory Data Analysis
- Feature Engineering
- LASSO Regression
- Model Evaluation (MAE, RMSE)
- Residual Diagnostics
- Data Visualization

## Key Insights

• The number of rooms (RM) shows strong positive correlation with housing price.  
• Percentage of lower status population (LSTAT) has a strong negative impact.  
• LASSO regression successfully removed less relevant variables.  
• Engineered features improved interpretability of housing market factors.
